# 八皇后問題

為了達到目的，任兩個皇后都不能處於同一條橫行、縱行或斜線上。<br>

# 真值表
![](images/truthtable.jpg)<br>
真值表是一種用於邏輯（特別是邏輯代數、布林函數和命題邏輯上）的數學用表，用來計算邏輯表示式在每種論證（即每種邏輯變數取值的組合）上的值。<br>
真值表可以用來判斷一個命題表示式是否對所有允許的輸入值皆為真，亦即是否為邏輯有效的。<br>
